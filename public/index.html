Paths

POST /email-checker

Returns a count of unique destination email addresses, following the gmail address filter rules.

Request Example
{
  "emails": [
    "test.email@gmail.com",
    "test.email+spam@gmail.com",
    "testemail@gmail.com"
  ]
}
200 OK
Response

200 OK with response

Response Example (200 OK) 
{
  "unique_email_addresses": 1
}

Schema Definitions

Response: object

unique_email_addresses: int

A list of the unique email destination addresses

Example
{
  "unique_email_addresses": 1
}
